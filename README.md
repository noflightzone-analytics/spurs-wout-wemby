# Spurs Without Wembanyama — Statistical Analysis
| 2025-26 Season

Full Python pipeline analysing every Spurs game without Victor Wembanyama.

## Requirements
pip install nba_api scikit-learn scipy matplotlib pandas numpy

## Usage
Set SKIP_FETCH=False for a live data pull, or True to reload from CSV.
Run all cells top to bottom in Google Colab or Jupyter.
